################# Git Rebase #############################
-> One of the code merging techniques
-> Always do on your own local/separate feature branch (ie. not the shared feature branch)
-> It helps to get the latest master branch code to your feature branch
Follow below step's
1. Do changes to your own current feature branch
2. To get the latest master branch changes to your current feature-branch

$your-feature-branch-name>
git checkout <<your-feature-branch-name>>

$your-feature-branch-name>
git status --> all uptodate

$your-feature-branch-name>
git rebase master <-- After this you will get all the lastest code from master branch to your -feature-branch.
