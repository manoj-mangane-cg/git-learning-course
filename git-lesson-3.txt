Git Fork : It helps to create exact replica of another Repo

Git rebase: 
$ git rebase -i HEAD~2 <--- by using this option you can correct the commit message context. Only updating commit-id
$ git log --oneline

Git Ammend Commends: When we need not to modify the last commit message and without this need to add the files to that commit use below commands
$ git add .
$ git commit --amend --no-edit
$ git log --oneline
